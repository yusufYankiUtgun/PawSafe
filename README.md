# PawSafe

A web app for reporting dangerous stray animals on an interactive map. Built as a course project for BIL 481 — Software Engineering.

## What it does

Users can drop pins on a map to mark locations of dangerous stray animals, upload photos as evidence, and validate or dispute each other's reports.
